# CIRO: The Heart of the City
## UI/UX Development Brief & Task Assignment

**Project Vision:** 
CIRO (Urban Crisis Response Agentic System) is a national-scale humanitarian orchestrator for Pakistan. It uses a multi-agent AI "Brain" to detect, analyze, and ethically prioritize crisis response (floods, heatwaves, fires, etc.) across the country.

---

### 1. Project Context & Aesthetic
We are building a **"Command Center"** experience. The interface must look and feel like a high-end, state-of-the-art government operations room.

*   **Theme:** Premium Dark Mode.
*   **Visual Language:** Glassmorphism (translucency, blur, depth).
*   **Keywords:** Agentic, Alive, Ethical, National, Responsive.
*   **Key Hook:** The **"Digital Twin"**—a real-time map visualization that predicts crisis outcomes and shows the life-saving impact of AI intervention.

---

### 2. Workflow & Security Policy
As we are in the high-fidelity design phase, the following workflow is in effect:
*   **Design-First:** All UI/UX work must be completed and approved in **Figma/Adobe XD** before any code is discussed.
*   **Access Control:** Access to the core Agentic Backend and Source Code is restricted to the Lead Integrator. You will provide standalone frontend components (HTML/CSS/TSX) which will then be integrated into the master system.
*   **Reporting:** All deliverables must be submitted directly to the Team Lead for review.

---

### 3. Immediate Task Assignments (Phase 1)

#### Task 1: The "Command Center" Design System
Create a unified visual language for CIRO. 
*   **Deliverable:** A Figma Style Guide including a custom HSL color palette (Dark Mode focused), Typography (Inter/Outfit), and Glassmorphism UI components (Cards, Buttons, Modals).
*   **Goal:** Establish the "Premium" feel immediately.

#### Task 2: Digital Twin Interaction Storyboard
Design the "WOW" factor of the app—the Inaction vs. Action slider.
*   **Deliverable:** High-fidelity mockups of the National Map. Show how the map changes visually when the user slides from "Predicted Disaster" (Red/Glitchy) to "Agentic Response" (Clean/Blue/Safe).
*   **Goal:** Make the impact of the AI visceral.

#### Task 3: "Agent Reasoning" Interface
The AI "thinks" before it acts. We need to visualize its logic.
*   **Deliverable:** A design for the "Ethical Trace" panel. This should show the agent's decision-making process (e.g., *"Prioritizing Karachi due to population density: +15% vulnerability"*).
*   **Goal:** Humanize the AI and build trust through transparency.

#### Task 4: Custom Crisis Iconography
*   **Deliverable:** A set of 6 custom icons (Flood, Heatwave, Medical, Fire, Civil Unrest, Agent Scan).
*   **Goal:** Ensure the app feels unique and not like a generic template.
